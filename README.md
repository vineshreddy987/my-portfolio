# Esukapalli Vinesh Reddy - Portfolio

## Portfolio Overview

This portfolio showcases Esukapalli Vinesh Reddy's work as an Aspiring AI Engineer. It features a professional showcase of machine learning projects, AI applications, and web development work.

## Features

- **Professional Profile**: Complete resume and education details
- **Project Showcase**: Featured AI, ML, and web development projects
- **Interactive UI**: Smooth animations and responsive design
- **Responsive Design**: Works seamlessly across all device sizes
- **Modern Aesthetic**: Clean design with professional styling

## Pages

1. **Home Page**: Main navigation hub
2. **About Page**: Professional background, education, and skills
3. **Projects Page**: Showcase of AI, ML, and web development projects

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Font Awesome
- **Animations**: CSS transitions and keyframe animations
- **Responsive Design**: Media queries for all device sizes

## Getting Started

To run this portfolio locally:

```bash
# Clone the repository
git clone https://github.com/vineshreddy987/my-portfolio.git

# Navigate to project directory
cd my-portfolio

# Open index.html in your browser
open index.html
```
